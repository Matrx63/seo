Chaîne de traitement pour Google News

Le principe a été décrit dans le cours : il s’agit de générer une liste de mots à utiliser pour créer un texte en accord avec un cluster de news proposé par Google News. Vous devez donc :

- Lire le flux RSS
- Extraire un cluster
- Calculer les fréquences des versions lemmatisées des n-grams contenus dans le cluster
- Réaliser un affichage
- Faire une interface de publication permettant de rédiger le texte, et de le mettre en ligne (titre et photo sont importants).
